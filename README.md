# ウッドデッキ床板計算ツール

ウッドデッキ・木道の施工時に必要な床板の幅調整を自動計算するWebアプリケーションです。

## 社内での使用方法

### 方法1: Vercelで公開（推奨・最も簡単）

**所要時間: 約5分**

1. **GitHubアカウントを作成**（持っていない場合）
   - https://github.com にアクセス
   - 無料でアカウント作成

2. **このプロジェクトをGitHubにアップロード**
   - GitHubで「New repository」をクリック
   - リポジトリ名: `deck-calculator`
   - 「Create repository」をクリック
   - このフォルダの全ファイルをアップロード

3. **Vercelでデプロイ**
   - https://vercel.com にアクセス
   - 「Sign Up」→「Continue with GitHub」
   - GitHubアカウントで連携
   - 「Import Project」→ 作成したリポジトリを選択
   - 「Deploy」をクリック

4. **完成！**
   - 数分でデプロイ完了
   - `https://deck-calculator-xxxxx.vercel.app` のようなURLが発行されます
   - このURLを社内で共有

**メリット:**
- 完全無料
- SSL対応（https://）
- 自動で最新版に更新可能
- スマホで「ホーム画面に追加」してアプリのように使える

---

### 方法2: Netlifyで公開

### 方法3: 社内サーバーで運用

**前提条件:**
- Node.js 16以上がインストールされていること

**手順:**

```bash
# 1. 依存関係をインストール
npm install

# 2. ビルド
npm run build

# 3. distフォルダを社内サーバーにアップロード
# distフォルダの中身を公開ディレクトリに配置
```

## スマホでアプリのように使う方法

### iPhoneの場合:
1. SafariでツールのURLを開く
2. 下部の共有ボタン📤をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面にアイコンが追加されます

### Androidの場合:
1. ChromeでツールのURLを開く
2. 右上のメニュー⋮をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面にアイコンが追加されます

---

## 使い方

1. **施工幅**: ウッドデッキの全体幅を入力（例: 3000mm）
2. **床板幅**: 定尺の床板幅を入力（例: 150mm）
3. **目地幅**: 床板間の隙間を入力（例: 5mm）
4. **最低床板幅**: カット後の最小許容幅を入力（例: 80mm）
5. 「計算する」ボタンをクリック

最大5つの施工案が表示され、調整が必要な板の枚数が少ない順に並びます。

## トラブルシューティング

### デプロイがうまくいかない場合
- package.jsonのファイルが正しくアップロードされているか確認
- Vercel/Netlifyのビルドログを確認

### 計算結果が表示されない
- 入力値が正しいか確認
- 最低床板幅が床板幅より小さいか確認
